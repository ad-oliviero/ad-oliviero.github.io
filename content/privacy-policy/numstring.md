# Num String

The application "Num String" ("Numero -> parola" in italian) is a program distributed on:

- Github:
    - [Desktop](https://github.com/ad-oliviero/num-string)
    - [Android](https://github.com/ad-oliviero/num-string-android)
- [Google play store](https://play.google.com/store/apps/details?id=com.tdb.numstring)

## Privacy policy

The desktop version does not collect any data from the user.

Same happens for the android version, but Google may collect some data.

For more information, see the [Google privacy policy](https://policies.google.com/privacy).
