An online tic-tac-toe game written in C with raylib, building for Linux, Windows and Android.
<!-- more -->
Simple TTT Online is a networked tic-tac-toe built in C on top of the raylib game library. A
single codebase compiles to Linux, Windows and Android, with the build steps and an Android
setup script included.

- Online multiplayer tic-tac-toe in C
- Rendered with raylib
- One codebase building for Linux, Windows and Android
